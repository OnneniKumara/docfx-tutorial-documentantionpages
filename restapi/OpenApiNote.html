OpenAPI Version Support

  • Currently (May 9, 2025), DocFX does not support OpenAPI 3.0 specifications.
  • For example, swagger.json files generated with OpenAPI 3.0 are not compatible.
  • OpenAPI 2.0 (swagger.json in OpenAPI 2 format) is supported.
  • Check the progress of this feature request here: https://github.com/dotnet/docfx/issues/2742.
